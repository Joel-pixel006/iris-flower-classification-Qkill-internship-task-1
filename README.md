# Iris Flower Classification

This project classifies iris flowers into 3 species:
- Setosa
- Versicolor
- Virginica

Process:
1. Load dataset
2. Visualize data
3. Split training and testing data
4. Train Decision Tree model
5. Evaluate using accuracy and confusion matrix

Tools Used:
- Python
- Pandas
- Matplotlib
- Scikit-learn
